利用 mathjax 使 hexo 支持 LaTex 公式

Posted on 2019-02-13

1. 使用 kramed 代替 marked

hexo默认的渲染引擎是marked，但是marked不支持mathjax。 kramed是在 marked的基础上进行修改。我们在工程目录下执行以下命令来安装kramed.

先删除marked

npm uninstall hexo-renderer-marked —save

然后安装kramed

npm install hexo-renderer-kramed —save

2. 不要使用 hexo-math

新版本的hexo在安装hexo-math的时候会出现警告信息，大致意思是说这个插件已经被弃用，所以不推荐使用hexo-math。
如果你已经安装 hexo-math, 请先卸载它：

npm uninstall hexo-math —save

然后安装 hexo-renderer-mathjax：

npm install hexo-renderer-mathjax —save
